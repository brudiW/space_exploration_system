# groundVector
Vektor mit der Länge = Erdradius,
Zeigt auf die Bodenposition des Orbiters (Ort wo der ˋlocationVectorˋ die Erdoberfläche schneidet)

# locationVector
Zeigt auf die Position des Orbiters in Relation zum Erdmittelpunkt (0, 0, 0)

# facingVector
Zeigt in Blickrichtung des Orbiters

# EarthGravity
